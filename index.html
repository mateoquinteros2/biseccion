¿Cómo insertar una función?

Tomando la función f(x) = x^2 - 2cos(x) – 1

Para ingresar x^2 deberá usarse el botón ^
En pantalla se mostrará por defecto Math.pow(x,1)
Donde x es la base y 1 es el exponente
El programa permite cambiar estos valores

Para ingresar 2cos(x) se escribirá 2 seguido de *
Se usará el botón cos(x) [lo mismo para sen(x) y tan(x)]
En pantalla se mostrará por defecto Math.cos(x)
Donde x es el valor para evaluar

Al final quedará así: Math.pow(x,2)-2*Math.cos(x)-1

Para ingresar Logaritmo Natural deberá usa el botón LN
En pantalla se mostrará Math.log(x)
Donde x es el numero que se desea evaluar

Para ingresar el número de Euler deberá usar el botón E
En pantalla se mostrará Math.E
Se puede reemplazar dentro de las variables de las demás botones

¿Cómo insertar los intervalos?

Los intervalos iniciales A y B pueden ser números
positivos o negativos. No se puede usar / para indicar
fracciones, se recomienda usar . (punto decimal)

¿Cómo insertar la tolerancia?

La tolerancia debe ser un numero positivo
Si es negativo el algoritmo lo pasa a positivo
Puede usar la expresión 1e-3 para indicar
0.001 o ingresar directamente 0.001
